# Tile Swaper Plugin for GB Studio

Replace tiles from a tileset in your scene, with support for multi-frame animations.

## 🎨 GB Tile Swapper — Builder Tool

Use the **GB Tile Swapper** visual tool to generate tile coordinates without doing any math:
load your level and tileset images, select tiles visually, copy the generated Tile Data,
and paste it directly into this plugin.

➜ **[GB Tile Swapper Tool](https://pixelpep.com/gbdev/gbtileswaper/)**

---

## Installation

1. Copy the `Tile Swaper` folder into your GB Studio project's `plugins/` folder
2. Create the `plugins/` folder at the root of your project if it doesn't exist
3. Restart GB Studio to load the plugin

---

## How to Use

### Step 1 — Register the tileset at scene start

Before using this plugin, GB Studio needs to know your tileset exists in the scene.

1. In your scene's **On Init** script, add an **empty GBVM Script** event at the very top
2. Inside it, click **Add Reference**
3. Choose **Tileset** and select your tileset PNG from the **Tilesets** folder of your project

This registers the tileset in memory for the scene. Without this step, the tile replacement will not work.

### Step 2 — Name your tileset file carefully

⚠️ **Avoid spaces, hyphens (`-`) and underscores (`_`) in your tileset filename.**

GB Studio uses these characters internally when generating GBVM symbol names, and they can cause conflicts or broken references. Use **CamelCase** or a simple lowercase word:

| ❌ Avoid | ✅ Use instead |
|----------|---------------|
| `my-tileset.png` | `mytileset.png` |
| `my_tileset.png` | `MyTileset.png` |
| `my tileset.png` | `tileset01.png` |

### Step 3 — Configure the plugin

Add a **Tile Swaper** event to your scene or actor script and fill in the fields:

**Tileset Name**
Enter the filename of your tileset PNG **without the extension**.
Example: if your file is `mytileset.png`, enter `mytileset`.

**Tile Data**
Paste the coordinates generated by the GB Tile Swapper tool.
Format: `x,y,tile;x,y,tile;`
Separate animation frames with a pipe `|`:
```
x,y,t;x,y,t;|x,y,t;x,y,t;|x,y,t;x,y,t;
```

**Frame Delay**
Number of game frames to wait between each animation frame.
Lower = faster, higher = slower. Range: 1–60.

**Animation Cycles**
How many times to repeat the full animation sequence.
- `0` = **Infinite loop** (runs until the scene ends)
- `1` = Play once
- `2+` = Repeat that many times

> ⚠️ For infinite loops (`0`), place this event inside an Actor's **On Update** script.

**Add REPAINT frame at end** *(optional)*
Adds a final frame that repaints the entire scene back to its original state after the animation completes.

## Tile Data Format

```
x,y,tile;x,y,tile;
```

- `x` — tile column on the scene map (0-based)
- `y` — tile row on the scene map (0-based)
- `tile` — tile index from the tileset (0-based)

**Single frame** (no animation):
```
1,9,80;2,9,81;
```

**Multi-frame animation** (frames separated by `|`):
```
1,9,80;2,9,81;|1,9,82;2,9,83;|1,9,84;2,9,85;
```

**Restore scene to original** (special keyword):
```
REPAINT
```

## Examples

### Replace tiles once (no animation)
```
1,9,80;2,9,81;
```

### 2-frame blinking light
```
1,9,80;|1,9,82;
```
Frame Delay: 15 — Cycles: 0 (infinite)

### 3-frame waterfall (2×2 tiles)
```
5,10,100;6,10,101;5,11,102;6,11,103;|5,10,104;6,10,105;5,11,106;6,11,107;|5,10,108;6,10,109;5,11,110;6,11,111;
```
Frame Delay: 8 — Cycles: 0 (infinite)

### Door opening (plays once)
```
10,8,50;|10,8,51;|10,8,52;|10,8,53;
```
Frame Delay: 4 — Cycles: 1

## Troubleshooting

**Tiles not appearing**
Make sure you registered the tileset with **Add Reference** in an empty GBVM Script at the start of your scene's On Init.

**Wrong tiles or missing symbols**
Check that your tileset filename has no spaces, hyphens, or underscores. Rename the file and re-add it to your project if needed.

**Animation not looping**
Set Cycles to `0` for infinite, and place the event inside an Actor's **On Update** script.

**Plugin not showing in GB Studio**
Make sure the `Tile Swaper` folder is inside your project's `plugins/` folder and restart GB Studio completely.
